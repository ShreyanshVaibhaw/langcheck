# Packaging

Per-user Windows installer and uninstaller — no administrator privileges, optional
start-at-login, background launch via `--background`. Built in delivery Step 12
(Installer and MVP Release). LangCheck contains no in-app updater and performs no
update checks; new versions are distributed as signed installers obtained outside
the application (see [`blueprint.md`](../../blueprint.md) Sections 21 and 21.2).
